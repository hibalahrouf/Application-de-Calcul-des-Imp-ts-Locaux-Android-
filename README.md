# 🏠 Application de Calcul des Impôts Locaux (Android)

##  Description
Cette application Android permet de calculer le montant des impôts locaux d’un bien immobilier en fonction de :

- La surface de la maison (en m²)
- Le nombre de pièces
- La présence ou non d’une piscine

L’application affiche automatiquement le montant total après saisie des données.

---

##  Fonctionnalités
- Saisie de la surface
- Saisie du nombre de pièces
- Option piscine (case à cocher)
- Calcul automatique de l’impôt
- Interface simple et intuitive

---

##  Logique de calcul
- Impôt de base = surface × 2  
- Supplément = (nombre de pièces × 50) + 100 si piscine  
- Total = impôt de base + supplément
